Although you can use the file explorer for managing files and directories, it is much faster from the command line. Not to mention, it's easier to use a command to force some actions than going through directory settings, such as deleting a directory that has Git installed. Git is a version control system that tracks changes in the files and directories of a project. If you were using file explorer, Git would not let you delete a directory that's being tracked. Either you would force the deletion, or delete git.

Using the Terminal

For an introduction to the terminoloy, go to the software section in Terminology. To come back to this page, either use the page nagivation, or menu, on the homepage, or click the link at the end of the software terminology section.

In your terminal, navigate to your desktop. The command to change directories is 'cd'. You can use the tilde symbol to change the root.

cd ~/Desktop

Try switching to other directories you know are in your computer like the Users directory, or the Downloads directory.

To see all the files and folders in your working directory, use the command: ls

ls

To open or navigate into one of the current directory's folders, you would cd into it with cd directory_name or cd ...directories/directory_name

The example ...directories/directory_name denotes a filepath. When you use file explorer, you can click on the display showing what folders you've entered, right click and copy the filepath. A filepath is like an address of directories to reach a file or folder. For example, C:/Users/GuestUser/Downloads, the absolute filepath to your Downloads folder.
